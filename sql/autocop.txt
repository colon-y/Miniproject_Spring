Google 크롬 지침
Google 크롬에서는 자동 완성 데이터를 끄고 싶을 뿐만 아니라 지우고 싶을 것입니다. 지침은 아래에 나열되어 있습니다.

Chrome에서 자동 완성 끄기

Chrome 메뉴 아이콘을 클릭합니다. (화면 우측상단 점3개)
설정을 클릭합니다.
"자동 완성" 섹션에서 자동 완성을 비활성화할 영역을 확장합니다.
켜져 있으면 설정을 OFF로 전환합니다. 시스템이 자동으로 설정을 저장합니다.
Chrome에서 자동 완성 데이터 지우기


Chrome 메뉴 아이콘을 클릭합니다. (화면 우측상단 점3개)
기록을 클릭한 다음 나타나는 메뉴에서 기록을 다시 클릭합니다.
검색 데이터 지우기를 선택합니다.
(또는 키보드에서 CTRL+SHIFT+DEL을 눌러 1-3단계를 건너뛸 수 있습니다.)
아직 선택하지 않은 경우 "고급" 탭을 클릭합니다.
상단에서 "모든 시간" 옵션을 선택하여 저장된 모든 데이터를 지웁니다.
"양식 데이터 자동 완성" 옵션이 선택되어 있는지 확인하십시오. "암호 및 기타 로그인 데이터"를 지우려면 해당 옵션도 선택해야 합니다.
"데이터 지우기"를 클릭합니다.
